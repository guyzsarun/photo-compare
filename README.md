# Photo Compare

A web application that allows users to upload and compare two photos side-by-side with interactive marker placement for precise point-to-point comparison.

## Features

- **Dual Image Upload**: Upload two images simultaneously for comparison
- **Side-by-Side View**: View both images aligned side-by-side for easy comparison
- **Interactive Markers**: Place numbered markers on both images to highlight and compare specific points
- **Synchronized Markers**: Markers are linked between images for easy reference
- **Zoom & Pan**: Zoom in/out and pan across images to examine details
- **Marker Management**: Add, remove, and edit marker labels
- **Export Comparison**: Save or export the comparison with markers intact
- **Responsive Design**: Works on desktop and tablet devices

## Use Cases

- **Before/After Comparisons**: Compare renovation, restoration, or editing work
- **Quality Control**: Identify differences in manufacturing or production
- **Medical Imaging**: Compare scans or x-rays over time
- **Design Reviews**: Compare design iterations or mockups
- **Photo Editing**: Evaluate editing changes and adjustments
- **Document Comparison**: Compare versions of documents or drawings
